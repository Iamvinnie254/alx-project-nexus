## FreshHarvest Marketplace - Complete Project Documentation

## Project overview

**FreshHarvest Marketplace** - A B2C e-commerce backend connecting Nairobi small-scale farmers directly to urban consumers for fresh produce. Eliminates middlemen, reduces waste, ensures fair pricing.

**Business Problem Solved**: Kenya's $1B+ online grocery market needs farm-fresh alternatives. 70% Nairobi residents want direct-from-farm produce.
